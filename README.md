## Description

Qualification work for RUDN University

Educational site on environmental and industrial safety management with elements of virtual immersion in a professional environment

## Project setup

```bash
$ npm install
```
Complete your .env file like in .env.example

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
